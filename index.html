Mentor @ Google Code In - Mifos Initiative (December 2014 - January 2015)

Google Code-in (GCI) is a contest for pre-university students (e.g., high school and secondary school students) with the goal of encouraging young people to participate in open source. We work with open source organizations, each of whom will provide a list of tasks to be completed by student contestants. Tasks can be anything an organization needs help with, from bug fixes, to writing code, to user experience research, to writing documentation. The contest takes place entirely online. As a mentor my responsibilities were :

  • To create Tasks with varying difficulty for students across the globe.
  • To answer queries of students related to the tasks they claim.
  • To review the Pull Requests and the Code submitted by the students.
  • To guide them on various topics that block a them while performing a task.
